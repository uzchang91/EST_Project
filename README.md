# Project Name
롯데웰푸드 사이트의 배너 페이지를 따라서 제작하는 정적 웹 프로젝트


## 📌 프로젝트 소개
이 프로젝트는 HTML/CSS 기반 UI 설계 실습을 목적으로 제작 되었습니다.
실제 서비스 구조를 모사하여 레이아웃 설계와 Tailwind CSS 프레임워크를 사용합니다.


## 🛠 Tech Stack
- HTML5
- CSS3 (Flex / Grid)
- Tailwind CSS
- Figma
